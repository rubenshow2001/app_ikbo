2. Crear un Proveedor (POST)


http://localhost/inventario_api/public/inventario/proveedor
Endpoint: POST /inventario/proveedor

Cuerpo de la Solicitud (JSON):
JSON

{
    "nombre_proveedor": "Lácteos del Campo S.A.",
    "contacto_proveedor": "Juan Pérez",
    "telefono_proveedor": "555-1234",
    "email_proveedor": "contacto@lacteoscampo.com",
    "direccion_proveedor": "Calle Falsa 123, Ciudad"
}



URL Base de la API (ejemplo): http://localhost/inventario_api/public

1. Realizar una Entrada de Inventario (POST)

Endpoint: POST /inventario/entrada

Cuerpo de la Solicitud (JSON):
JSON

{
    "sku": "LCHENT001",
    "numero_lote": "LOTE20250718-A",
    "cantidad": 100,
    "fecha_vencimiento": "2026-12-31",
    "id_proveedor": 1,
    "ubicacion_almacen": "Bodega Principal - Estante 5",
    "costo_compra_unitario": 10.50
}






2. Realizar una Salida de Inventario (POST)

Endpoint: POST /inventario/salida

Cuerpo de la Solicitud (JSON):
JSON

{
    "sku": "PROD001",
    "cantidad": 30,
    "razon_movimiento": "Venta a cliente X"
}




3. Listar Productos (GET)

Endpoint: GET /inventario/productos

Ejemplo cURL:
Bash

curl -X GET \
  http://localhost/inventario_api/public/inventario/productos
